## Project setup
1. Get the repository
2. Go to the root of the repository and run `$ bundle exec pod install`
3. Enjoy

## Frameworks:
1. Network: Alamofire
2. Utilities: SDWebImage, SVProgressHUD
3. Database: RealmSwift
